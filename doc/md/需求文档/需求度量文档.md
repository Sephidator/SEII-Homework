# 2. 加权因子说明
	因为本系统（灯具行业进销存管理系统）属于中等系统，所以功能点测度的加权因子：
	输入数量加权因子 = 4，
	输出数量加权因子 = 5，
	查询数量加权因子 = 4，
	逻辑文件加权因子 = 10，
